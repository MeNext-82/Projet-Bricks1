◆ game_add_bonus()

void game_add_bonus ( game * g,
double x,
double y,
bool down )

Ajoute un bonus au jeu, en le créant à la position (x, y).

Remarques
La fonction doit allouer un nouveau bonus et l'ajouter au tableau bonus_list du jeu. Si le tableau bonus_list est plein (c'est à dire si nb_bonus est égal à size_bonus), la fonction doit doubler la taille du tableau bonus_list pour y faire de la place avant d'ajouter le nouveau bonus.
Paramètres
gLe jeu.
xLa position horizontale du bonus sur le plateau de jeu.
yLa position verticale du bonus sur le plateau de jeu.
downIndique si le bonus doit descendre (true) ou monter (false)

◆ game_remove_bonus()

void game_remove_bonus ( game * g,
int i )

Supprime le bonus d'indice i du jeu.

La fonction doit libérer la mémoire occupée par le bonus, le retirer du tableau bonus_list. Pour combler le trou, on réorganisera le tableau bonus_list de manière en remplaçant le bonus supprimé par le dernier bonus du tableau (celui d'indice nb_bonus - 1). Enfin, la fonction doit décrémenter le nombre de bonus nb_bonus (on a retiré un bonus du jeu).

Paramètres
gLe jeu.
iL'indice du bonus à supprimer dans le tableau bonus_list.

◆ game_capture_bonus()

bool game_capture_bonus ( game * g,
paddle * p )

Teste si une raquette touche un bonus présent sur le plateau de jeu et change l'état du jeu en conséquence (activation du bonus, suppression du bonus, etc.).

Si une raquette touche un bonus, le bonus est activé et supprimé du plateau de jeu. L'effet du bonus dépend de son type (voir la structure bonus dans game_geometry.h).

Renvoie
true si la raquette touche un bonus, false sinon.
Paramètres
gLe jeu.
pLa raquette qui pourrait toucher un bonus.
