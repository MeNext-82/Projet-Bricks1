Description détaillée

Structure représentant un tableau de scores. Elle contient les noms des joueurs, leurs scores, la taille du tableau et l'indice du score du joueur actuel (s'il est présent).

Cette structure est utilisée dans trois situations spécifiques :

  • Quand on affiche les scores dans le menu. Dans ce cas, elle est remplie avec les scores chargés depuis le fichier de scores uniquement (il n'y a pas de score du joueur actuel à afficher dans le menu).
  • Quand on affiche le score à la fin d'un jeu Pong. Dans ce cas, elle contient les scores chargés depuis le fichier de scores plus le score du joueur actuel.
  • Quand on affiche le score final à la fin d'un jeu briques. Dans ce cas, elle contient les scores chargés depuis le fichier de scores plus le score du joueur actuel.
Remarques
Le tableau des noms des joueurs contient des chaînes de caractères de longueur maximale 64 (y compris le caractère de fin de chaîne). Cela signifie que les noms des joueurs peuvent comporter au maximum 63 caractères.
