Description détaillée

Gestion des meilleurs scores en mode classique. Contient une fonction pour charger, enregistrer et trier les meilleurs scores du fichier de scores.

Vous devez respecter le format spécifié pour le fichier de scores afin de garantir un bon chargement et sauvegarde des scores (voir la documentation du projet pour plus de détails).

Appelée par l'interface pour charger un tableau de scores depuis un tableau de chaînes de caractères extrait d'un fichier de scores. Si un player_score est fourni, elle l'ajoute comme une entrée pour le joueur actuel sous le nom "Player".

Renvoie

Paramètres
inputTableau de chaines de caractères contenant les lignes lues depuis le fichier de scores.
sizeTaille du tableau de chaines de caractères (0 si le fichier de scores n'existe pas ou est vide).
player_scorePointeur vers le tableau de scores du joueur actuel (NULL quand utilisé pour charger les meilleurs scores depuis le menu).

◆ get_pong_winner_score()

bool get_pong_winner_score ( game * g,
short p,
int * score_array )

Appelée par l'interface pour récupérer les scores du joueur gagnant à la fin d'une partie de Pong.

Le tableau score_array pris en paramètre est supposé être de taille SC_SIZE. On doit y écrire les scores du joueur gagnant:

  • Si le paramètre p est égal à 0 ou 1, la fonction récupère les scores du joueur correspondant (0 pour le joueur 1, 1 pour le joueur 2) et les écrit dans score_array. Si ce joueur n'est pas présent dans la partie, la fonction retourne false.
  • Sinon (typiquement si p est égal à -1), la fonction retourne false.
Paramètres
gPointeur vers la structure de jeu.
pNuméro du joueur gagnant (0 ou 1). Sinon, la fonction retourne false.
