Description détaillée

Ce module gère les débris qui apparaissent à l'écran lorsqu'une brique est détruite. Les débris sont de petits rectangles qui se déplacent dans des directions aléatoires à partir de la position de la brique détruite, et disparaissent après un certain nombre de frames.

Remarques
Les fonctions de ce module sont déjà écrites, vous n'avez pas à les implémenter vous même. Vous devez cependant appeler add_debris() dans la fonction game_remove_brick() du module game_bricks.c pour ajouter des débris à chaque fois qu'une brique est détruite.
