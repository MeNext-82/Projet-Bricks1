Description détaillée

Structure représentant l'état du jeu, y compris le plateau de jeu, les briques, les balles, les raquettes.

Remarques
Le plateau est composé de nb_columns x nb_rows cellules. Chaque cellule est de taille CELL_WIDTH x CELL_HEIGHT et peut contenir une brique (seulement en mode Briques) ou être vide. Le plateau de jeu entier est donc de taille (CELL_WIDTH * nb_columns) x (CELL_HEIGHT * nb_rows).
