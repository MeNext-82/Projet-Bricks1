\doxysubsection{Description détaillée}
Ce module gère les débris qui apparaissent à l\textquotesingle{}écran lorsqu\textquotesingle{}une brique est détruite. Les débris sont de petits rectangles qui se déplacent dans des directions aléatoires à partir de la position de la brique détruite, et disparaissent après un certain nombre de frames. 

\begin{DoxyRemark}{Remarques}
Les fonctions de ce module sont déjà écrites, vous n\textquotesingle{}avez pas à les implémenter vous même. Vous devez cependant appeler \doxylink{game__debris_8h_a8f4925ee142dc8c6023e33093e190eff}{add\+\_\+debris()} dans la fonction \doxylink{game__bricks_8h_aff980d4420910e1eb6b530350f10185d}{game\+\_\+remove\+\_\+brick()} du module game\+\_\+bricks.\+c pour ajouter des débris à chaque fois qu\textquotesingle{}une brique est détruite. 
\end{DoxyRemark}
