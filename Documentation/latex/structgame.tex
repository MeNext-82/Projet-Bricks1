\doxysubsection{Description détaillée}
Structure représentant l\textquotesingle{}état du jeu, y compris le plateau de jeu, les briques, les balles, les raquettes. 

\begin{DoxyRemark}{Remarques}
Le plateau est composé de nb\+\_\+columns x nb\+\_\+rows cellules. Chaque cellule est de taille \doxylink{game_8h_ae1dbb0e959fde337fa4e436b61bb2bac}{CELL\+\_\+\+WIDTH} x \doxylink{game_8h_a8519cbc2997bb2eb21cf5571b37b38bd}{CELL\+\_\+\+HEIGHT} et peut contenir une brique (seulement en mode Briques) ou être vide. Le plateau de jeu entier est donc de taille (\doxylink{game_8h_ae1dbb0e959fde337fa4e436b61bb2bac}{CELL\+\_\+\+WIDTH} \texorpdfstring{$\ast$}{*} nb\+\_\+columns) x (\doxylink{game_8h_a8519cbc2997bb2eb21cf5571b37b38bd}{CELL\+\_\+\+HEIGHT} \texorpdfstring{$\ast$}{*} nb\+\_\+rows). 
\end{DoxyRemark}
