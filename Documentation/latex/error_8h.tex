\doxysubsection{Description détaillée}
Macros permettant d\textquotesingle{}afficher des messages d\textquotesingle{}erreur. 

Le fichier fournit les macros \doxylink{error_8h_a3067238f546cd726d1c6097bf36766b7}{TRACE()}, \doxylink{error_8h_a4d9c2becab71a33443aeccb038726b1b}{DEBUG()}, \doxylink{error_8h_a84184e0f8203a79696f66122db589780}{INFO()}, \doxylink{error_8h_a8812bb69ff94adee37e14e8371d80124}{WARNING()}, \doxylink{error_8h_af75f6bd3a08e61c05dc8d9aeba74149a}{ERROR()}, \doxylink{error_8h_abb066e119222d55f4d8213bb87ba9f38}{CRITICAL()} et \doxylink{error_8h_a9841d8ca1b3695fac54a05bbd7f37af8}{FATAL()}, qui permettent d\textquotesingle{}afficher des messages d\textquotesingle{}erreur sur la sortie d\textquotesingle{}erreur, et, pour la macro \doxylink{error_8h_a9841d8ca1b3695fac54a05bbd7f37af8}{FATAL()}, de terminer le programme. Les messages d\textquotesingle{}erreur sont affichés avec le nom de la fonction, le nom du fichier et le numéro de la ligne où l\textquotesingle{}appel à la macro est effectué.

Les autres macros du fichier sont essentiellement destinées à implémenter les macros \doxylink{error_8h_a3067238f546cd726d1c6097bf36766b7}{TRACE()}, \doxylink{error_8h_a4d9c2becab71a33443aeccb038726b1b}{DEBUG()}, \doxylink{error_8h_a84184e0f8203a79696f66122db589780}{INFO()}, \doxylink{error_8h_a8812bb69ff94adee37e14e8371d80124}{WARNING()}, \doxylink{error_8h_af75f6bd3a08e61c05dc8d9aeba74149a}{ERROR()}, \doxylink{error_8h_abb066e119222d55f4d8213bb87ba9f38}{CRITICAL()} et \doxylink{error_8h_a9841d8ca1b3695fac54a05bbd7f37af8}{FATAL()}. Elles ne sont pas destinées à être utilisées directement par l\textquotesingle{}utilisateur (mais rien ne l\textquotesingle{}empêche).

Les macros \doxylink{error_8h_a3067238f546cd726d1c6097bf36766b7}{TRACE()}, \doxylink{error_8h_a4d9c2becab71a33443aeccb038726b1b}{DEBUG()}, \doxylink{error_8h_a84184e0f8203a79696f66122db589780}{INFO()}, \doxylink{error_8h_a8812bb69ff94adee37e14e8371d80124}{WARNING()}, \doxylink{error_8h_af75f6bd3a08e61c05dc8d9aeba74149a}{ERROR()}, \doxylink{error_8h_abb066e119222d55f4d8213bb87ba9f38}{CRITICAL()} et \doxylink{error_8h_a9841d8ca1b3695fac54a05bbd7f37af8}{FATAL()} prennent en argument un message à afficher, qui peut contenir des spécificateurs de conversion comme dans la fonction printf. Elles peuvent prendre des arguments supplémentaires, un par spécificateur de conversion.

\begin{DoxyNote}{Note}
Les messages d\textquotesingle{}un certain type sont affichés en fonction du niveau de debug, fixé par la variable \doxylink{error_8h_ac2d33ccaf63f5d5b66552b95426c0137}{DEBUG\+\_\+\+LEVEL}. Par exemple, si le niveau de debug est à \doxylink{error_8h_af75f6bd3a08e61c05dc8d9aeba74149a}{ERROR}, seules les macros \doxylink{error_8h_af75f6bd3a08e61c05dc8d9aeba74149a}{ERROR()}, \doxylink{error_8h_abb066e119222d55f4d8213bb87ba9f38}{CRITICAL()} et \doxylink{error_8h_a9841d8ca1b3695fac54a05bbd7f37af8}{FATAL()} afficheront leur message. Les autres macros n\textquotesingle{}afficheront rien. Par défaut, le niveau de debug est \doxylink{error_8h_a84184e0f8203a79696f66122db589780}{INFO}.~\newline
~\newline
 {\bfseries{Niveaux de debug :\+}}~\newline

\begin{DoxyItemize}
\item \doxylink{error_8h_a4ac568348fbda3f2ce37f4cfc14c9598aac132f2982b98bcaa3445e535a03ff75}{OFF} (seulement les messages FATAL sont affichés),
\item \doxylink{error_8h_abb066e119222d55f4d8213bb87ba9f38}{CRITICAL} (seulement les messages CRITICAL et FATAL),
\item \doxylink{error_8h_af75f6bd3a08e61c05dc8d9aeba74149a}{ERROR} (seulement les messages ERROR, CRITICAL et FATAL),
\item \doxylink{error_8h_a8812bb69ff94adee37e14e8371d80124}{WARNING} (seulement les messages WARNING, ERROR, CRITICAL et FATAL),
\item \doxylink{error_8h_a84184e0f8203a79696f66122db589780}{INFO} (tous les messages sauf DEBUG et TRACE),
\item \doxylink{error_8h_a4d9c2becab71a33443aeccb038726b1b}{DEBUG} (tous les messages sauf TRACE),
\item \doxylink{error_8h_a3067238f546cd726d1c6097bf36766b7}{TRACE} (tous les messages).
\end{DoxyItemize}

Pour changer le niveau de debug, il faut définir la macro \doxylink{error_8h_ac2d33ccaf63f5d5b66552b95426c0137}{DEBUG\+\_\+\+LEVEL} avant d\textquotesingle{}inclure \doxylink{error_8h}{error.\+h}, typiquement à la compilation. Par exemple, pour afficher tous les messages de debug, il faut définir \doxylink{error_8h_ac2d33ccaf63f5d5b66552b95426c0137}{DEBUG\+\_\+\+LEVEL} à \doxylink{error_8h_a3067238f546cd726d1c6097bf36766b7}{TRACE}. Pour cela, on peut compiler le programme ainsi :\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{make\ -\/B\ CFLAGS=\textcolor{stringliteral}{"{}-\/DDEBUG\_LEVEL=TRACE"{}}}

\end{DoxyCode}
